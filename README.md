# Spam Email Detection

---

## 📖 Project Overview

In this task, I built a beginner-level Machine Learning model using Scikit-Learn to classify emails as Spam or Not Spam.

The implementation is done in a Jupyter Notebook, and the dataset is provided separately as a CSV file.

The main purpose of this project is to understand the basic workflow of a Machine Learning classification model.

---

## 🎯 Objective

- To learn the basics of Machine Learning
- To understand classification problems
- To build a predictive model using Scikit-Learn
- To evaluate model performance using standard metrics

---

## 🛠 Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-Learn  
- Jupyter Notebook

---

## 🔄 Steps Performed

The following steps are implemented in the Jupyter Notebook:

1. Importing required libraries  
2. Loading the dataset using Pandas  
3. Basic data exploration  
4. Data preprocessing  
5. Splitting data into training and testing sets  
6. Training the model (Logistic Regression / Naive Bayes)  
7. Evaluating the model using:
   - Accuracy Score  
   - Confusion Matrix  
   - Classification Report  
8. Simple visualization of results  

---

## 📊 Model Evaluation

The model performance is evaluated using:

- Accuracy Score – To check overall prediction accuracy  
- Confusion Matrix – To understand correct and incorrect predictions  
- Classification Report – To analyze precision, recall, and F1-score  

---

## 🎓 Learning Outcomes

Through this project, I learned:

- How to prepare and clean data  
- How to split data into training and testing sets  
- How to train a classification model  
- How to evaluate model performance  
- Basic understanding of predictive modeling
